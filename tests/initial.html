Max Media Manager Testing Suite

In order to run tests, please copy the test.conf.ini file in the Max /etc directory into the Max /var directory, and edit the file, so that it contains your database server details, and your webpath access details.

Please also ensure that the web server user has permission to write to the Max /var and Max /var/plugins directories.

You will also need to give your web server user permission to write to the Max /tests/results directory if you want to run the visualisation tests.
