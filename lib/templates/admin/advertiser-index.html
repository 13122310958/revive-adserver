Notes to developer:

  1. The adnetwork campaign can be in the following states:
    • Pending - adnetwork has not reviewed the publisher site yet (corresponding CSS class: .sts-pending)
    • Rejected - adnetwork has rejected site (corresponding CSS class: .sts-rejected)
    • Running - adnetwork has accepted site (corresponding CSS class: .sts-accepted)
  2. Adnetwork campaigns cumulative status row (the row with adnetwork name) rules (should be shown only if advertiser section is collapsed) :
    • if status is the same for all adnetwork campaigns, show status name prepended with "All", omit all if there is only one campaign, see TradeDoubler and TexLinkAds examples
    • if statuses are different, show the number of sites with the first available of the following statuses "Rejected", "Running", "Pending", see CPX example
  3. The advertiser signed-up campaign can be in one of the following states:
    • Awaiting approval - new campaign, needs user action (corresponding CSS class: .sts-awaiting)
    • Rejected - campaign has been rejected (corresponding CSS class: .sts-rejected)
    • Running - campaign has been accepted and is currently running (corresponding CSS class: .sts-accepted)
    • Paused - campaign has been accepted but currently is paused and is not running (corresponding CSS class: .sts-paused)
    • Not started yet - campaign has been accepted but the start date has not been reached yet (corresponding CSS class: .sts-not-started)
    • Finished - campaign has been accepted and is currently finished (corresponding CSS class: .sts-finished)
  4. Advertiser sign-up campaigns cumulative status row (the row with advertiser name) rules (should be shown only if advertiser section is collapsed)
    • if status is the same for all adnetwork campaigns, show status name prepended with "All", omit "All" if there is only one campaign, in case of single "Awaiting approval" the cumulative status should be a link to campaign edit, see Coca Cola example, Monster Job, Dell
    • if statuses are different, show the number of sites with the first available of the following statuses "Awaitng approval", "Running", "Not started yet", "Paused", Rejected", see Blackberry and Nike example
